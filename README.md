# listing-searcher

Find Listing Ads URL (Japanese).

[![MIT License](http://img.shields.io/badge/license-MIT-blue.svg?style=flat)](LICENSE)

## Usage

### Step1

```bash
$ npm install
```

### Step2

Please edit list.txt with the words you want to search.

### Step3

```bash
$ npm run start
```

or

```bash
$ node index.js
```
